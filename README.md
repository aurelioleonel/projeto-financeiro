O projeto **Financeiro 2.0** foi desenvolvido durante as aulas do professor Heber.
Este projeto tem como objetivo simular uma agenda de gastos para mobile. 

###  Tecnologias 
* Delphi - Pascal
* Firemonkey - Mobile
* SQLite - banco de dados
